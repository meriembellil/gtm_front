Module Display

menu: Display

path: /display

description: ce module permet la gestion des "Display" 

- index contient la liste des "Display"
- l'utilisateur a la possibilité de voir le détail d'un "Display"
- l'utilisateur a la possibilité de modifier ou supprimer un display d'aujord'hui
- le dossier detailComponents contient les components du detail
- le dossier updateComponents contient les components de la modification
- le dossier createComponents contient les components de la creation d'un "Display" => cette action a été annuler dans ce module et ajouter au planning
